is_floatable(s)

Input: String or number
Output: Boolean

Checks if a given value can be converted to a float.

is_floatable("10.34")  # True
is_floatable("pepe")  # False

TableCsv.createRow(rawRow)

Input: String
Output: Dictionary

Creates and returns a row dictionary from a raw string input.

createRow("10,20,30")  # {'id': 10.0, 'c1': 20.0, 'c2': 30}

TableCsv.__init__(csvFile, splitterType, progress=False, rowLimit=9999999999)

Input: File path, String, Boolean, Integer
Output: None

Initializes the TableCsv object by reading and parsing the CSV file.

tbl = TableCsv('example.csv', ',', progress=True)

TableCsv.printItemTypes()

Input: None
Output: None

Prints all headers of the table.

tbl.printItemTypes()

TableCsv.getItemTypes()

Input: None
Output: List

Returns a list of all headers.

headers = tbl.getItemTypes()

TableCsv.getRow(n)

Input: Integer
Output: Dictionary

Returns the row at index n.

row = tbl.getRow(1)

TableCsv.getRowsByValue(htag, value)

Input: String, Any
Output: List of dictionaries

Returns rows where the specified column has the specified value.

rows = tbl.getRowsByValue('id', 2)

TableCsv.getColumnList(column)

Input: String
Output: List

Returns all values from the specified column.

values = tbl.getColumnList('id')

TableCsv.printRows()

Input: None
Output: None

Prints all rows in the table.

tbl.printRows()

TableCsv.rowAppend(rr)

Input: String
Output: None

Adds a new row to the table.

tbl.rowAppend("5,20,Another row")

TableCsv.rowDefenestrateByValue(htag, vall)

Input: String, Any
Output: None

Deletes rows where the specified column has the specified value.

tbl.rowDefenestrateByValue('id', 2)

TableCsv.rowLeaveByValue(htag, vall)

Input: String, Any
Output: None

Keeps only rows where the specified column has the specified value.

tbl.rowLeaveByValue('id', 2)

TableCsv.rowDefenestrateN(index)

Input: Integer
Output: None

Removes the row at the specified index.

tbl.rowDefenestrateN(1)

TableCsv.saveTo(flname)

Input: String
Output: None

Saves the table to the specified CSV file.

tbl.saveTo('newfile.csv')

TableCsv.save()

Input: None
Output: None

Saves the table to the original CSV file.

tbl.save()

TableCsv.maxID()

Input: None
Output: Integer

Returns the maximum ID value from the table.

max_id = tbl.maxID()

CreateCsvTable(pth, hdrs, overwrite=None, progress=False)

Input: String, List of strings, Boolean, Boolean
Output: TableCsv object

Creates a new CSV file and returns a TableCsv object.

tbl = CreateCsvTable('new_table.csv', ['id', 'name', 'value'], overwrite=True)
